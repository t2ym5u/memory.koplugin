# memory.koplugin

A Memory (Concentration) card game plugin for [KOReader](https://github.com/koreader/koreader).

## Rules

Find all matching pairs by flipping two cards at a time. Matching cards stay face up; non-matching cards flip back. Remember card positions to make matches efficiently. Win by revealing all cards.

## Features

- **Multiple grid sizes** — 4×4, 6×6 and more
- **Various card sets** — numbers, symbols, icons
- **Move counter** — tracks the number of attempts
- **Auto-save** — game state saved and restored on next launch

## Installation

1. Download `memory.koplugin.zip` from the [latest release](../../releases/latest).
2. Extract into the `plugins/` folder of your KOReader data directory.
3. Restart KOReader.
4. Open the menu → **Tools** → **Memory**.

## Controls

| Action | How |
|--------|-----|
| Flip a card | Tap it |
| New game | Tap **New** |
| Change grid size | Tap **Grid** |
| Show rules | Tap **Rules** |
